# Privacy Policy for Question Bot
Last Updated: 07/05/2026

Thank you for using [Bot Name]! This Privacy Policy explains what data we collect, why we collect it, and how it is stored when you invite our bot to your Discord Server.

### 1. Data We Collect and Store
To function properly, the bot stores the following data in a secure cloud database:
- **Server IDs (Guild IDs):** To identify which server is requesting a response.
- **Channel IDs:** We store the IDs of the channels configured during the `/setup` command (Questions channel, Knowledge channel, Missing Answers channel).
- **API Keys:** If you provide a third-party API key (e.g., Google Gemini), it is stored securely in our database solely to process AI requests for your specific server. 

### 2. Data We Process But Do NOT Store
- **User Messages:** When a user asks a question in the designated questions channel, the text of the message is temporarily sent to the Google Gemini API to generate a response. **We do not save, log, or store user chat history in our database.**
- **Knowledge Channel Text:** Text typed into the designated "Knowledge Channel" is temporarily cached in the bot's live memory to answer questions. 

### 3. Third-Party Services
Because this bot utilizes AI, messages sent in the configured channels are processed by **Google's Gemini API**. By using this bot, you also agree to Google's API Terms of Service. 

### 4. Data Removal
If you wish to have your server's data (including your API key and Channel IDs) permanently deleted from our database, simply kick the bot from your server or contact us.
